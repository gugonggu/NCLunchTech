# 룰렛 휠 UI 재설계

- 후보 이름을 원형 섹터에 배치한 CSS 룰렛으로 표시한다.
- 고정 상단 포인터와 중앙 버튼을 두고, 서버가 이미 선택한 후보의 섹터가 포인터 아래에서 멈추도록 4~6회전 감속 애니메이션을 적용한다.
- 후보가 너무 많으면 시각 섹터는 최대 8개를 표시하고 전체 후보는 기존 추천 로직에 그대로 사용한다.
- 결과 액션은 모든 화면 폭에서 동일한 2열 grid로 배치한다. 추천 화면의 룰렛 진입도 grid 항목으로 배치해 좁은 화면에서 눌리거나 세로로 깨지지 않게 한다.
- 새 라이브러리, DB 변경, 추천 선정 규칙 변경은 없다.
